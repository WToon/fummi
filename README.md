# SeedBox

Fundamentals of human computer interaction project

## Getting Started

Clone the repository

### Prerequisites

 Node.js
 
### Installing

1. Open both folders in a terminal
2. Run npm install to install the required dependencies

## Local development

1. Open 'oAuthLocal' in a terminal and run 'npm start'. This terminal will run the local authentication server.
2. Open 'seedbox' in a terminal and run 'npm start'. This terminal will run the development server for the app.
