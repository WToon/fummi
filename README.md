# SeedBox

Fundamentals of HCI project

## Getting Started

Clone the repository

### Prerequisites

 Node.js
 
### Installing

1. Open both folders in a terminal
2. Run ```npm install``` to install the required dependencies

## Local development

1. Open the ```oAuthLocal``` directory in a terminal and run ```npm start```. This terminal will run the local authentication server.
2. Open the ```seedbox``` directory in a terminal and run ```npm start```. This terminal will run the development server for the app.
3. You can now view *seedbox* in the browser by visiting the URL as seen in the ```seedbox``` terminal (http://localhost:3000/)
